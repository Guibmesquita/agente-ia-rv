# Roteiro de Testes End-to-End (E2E) — Aplicação SVN

## Objetivo

Validar todas as funcionalidades do aplicativo, garantindo:

* Funcionamento completo de cada aba
* Fluxos críticos operacionais
* Upload e processamento de conhecimento
* Busca do agente
* Campanhas de WhatsApp (individual e em massa)
* Envio de mídia (imagem, PDF, Excel)
* UX, UI e não regressão visual

---

## Pré-requisitos

* Ambiente rodando com frontend e backend conectados
* Número de WhatsApp para testes configurado:

  * 11947033973
* Acesso a todas as abas
* Possibilidade de upload de arquivos

# 1. Login e Navegação Geral

## 1.1 Login

* Acessar a tela de login
* Efetuar login com usuário válido
* Verificar:

  * Redirecionamento correto
  * Sidebar carregada
  * Nenhum erro visual
  * Layout consistente

## 1.2 Sidebar e Navegação

* Clicar em todas as opções do menu lateral
* Verificar:

  * Estado ativo correto
  * Nenhum menu quebrado
  * Submenus abrindo corretamente (Configurações)
  * Espaçamento entre blocos (principal vs configurações)

# 2. Aba Conversas (WhatsApp)

## 2.1 Lista de Conversas

* Acessar aba Conversas
* Verificar:

  * Conversas carregam
  * Conversa mais recente no topo
  * Scroll funcional
  * Nenhum layout quebrado

## 2.2 Abrir Conversa

* Clicar em uma conversa existente
* Verificar:

  * Histórico completo aparece
  * Scroll para última mensagem
  * Nome do contato correto

## 2.3 Enviar Mensagem de Texto

* Enviar uma mensagem de texto
* Verificar:

  * Mensagem aparece imediatamente no UI
  * Mensagem é enviada via WhatsApp
  * Não há duplicação
  * Estado de loading (se existir) funciona

## 2.4 Receber Mensagem

* Enviar mensagem do celular para o número do bot
* Verificar:

  * Mensagem aparece no UI sem refresh
  * Conversa sobe para o topo
  * Badge de nova mensagem (se existir)

## 2.5 Enviar Imagem

* Enviar imagem pelo frontend
* Verificar:

  * Upload funciona
  * Mensagem aparece no chat
  * Imagem chega no WhatsApp
  * Preview correto no UI

## 2.6 Enviar Arquivo (PDF ou Excel)

* Enviar PDF ou XLS pelo frontend
* Verificar:

  * Upload funciona
  * Nome do arquivo aparece
  * Arquivo chega no WhatsApp
  * Link/download funcional no UI

## 2.7 Receber Áudio

* Enviar áudio do WhatsApp para o bot
* Verificar:

  * Áudio aparece no chat
  * Player funciona no UI
  * Não quebra layout
